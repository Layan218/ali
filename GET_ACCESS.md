# How to Get Access to Firebase & Vercel

## üîê Firebase Credentials

### Option 1: Get from Vercel (Easiest)
1. Go to: https://vercel.com/dashboard
2. Sign in with your GitHub account
3. Find project: **"ali"** or **"appttt"**
4. Click on the project
5. Go to: **Settings** ‚Üí **Environment Variables**
6. Copy all `NEXT_PUBLIC_FIREBASE_*` values
7. Paste them into your `.env.local` file

### Option 2: Ask Team to Add You to Firebase
**You need to ask a team member who has admin access to:**
1. Go to: https://console.firebase.google.com/
2. Select the Firebase project
3. Go to: **Project Settings** ‚Üí **Users and permissions**
4. Click **Add member**
5. Enter your email address
6. Give you **Editor** or **Viewer** role
7. Once added, you can access the Firebase console and get credentials

### Option 3: Ask Team to Share Credentials
Ask a team member to:
- Share the Firebase config values with you securely
- Or provide access to a shared password manager/document

## üåê Deployment URL

### Option 1: Get from Vercel Dashboard
1. Go to: https://vercel.com/dashboard
2. Find your project
3. The deployment URL is shown at the top of the project page
4. It will look like: `https://ali-xxxxx.vercel.app` or a custom domain

### Option 2: Ask Team to Add You to Vercel
**You need to ask a team member who has admin access to:**
1. Go to: https://vercel.com/dashboard
2. Find the project: **"ali"** or **"appttt"**
3. Go to: **Settings** ‚Üí **Team**
4. Click **Invite**
5. Enter your email address
6. Give you **Member** or **Developer** role
7. Once added, you'll have access to the dashboard and can see the URL

### Option 3: Ask Team for the URL
Ask a team member to share the deployment URL with you
